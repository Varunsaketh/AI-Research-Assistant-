This is a basic Multi-agent system which consists of 3 agents 
### 1. Research Agent:
Agent which collects the data from the web or existing database using gemini LLM and tools like SerperDevTool()
### 2. Analysis Agent:
Agent which analizes from the scrapped data 
### 3. Writer Agent:
Agent which writes from the analyzed data 

better to use 3 different LLM for three duifferent tasks to reduce token count
